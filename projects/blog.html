In this project, we will use modules such as Express to write code, use body parser to get user input values, and use Mongoose to connect to MongoDB database. Then we create different routes for different purposes, such as, for example, a Get route to get a blog, a Post route to publish a blog, and a Delete route to delete a blog.

SQL & NOSQL

NoSQL databases (aka "more than just SQL") are non-tabular databases that store data differently than relational tables. There are different types of NoSQL databases based on the data model. The main types are Document, Key Value, Wide Column and Chart. They offer flexible schemas and can easily scale with large amounts of data and heavy user loads.
